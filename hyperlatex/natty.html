How To Install Hyperlatex on
Natty Narwhal (Ubuntu 11.04)

written by: John C. Lusth

Revision Date: August 8, 2012

Printable Version

Installation Notes

Follow these steps:

  • Remove emacs23, if present: 
        sudo apt-get remove emacs23
  • Install xaw3dg: 
        sudo apt-get install xaw3dg
  • Download these packages:
    • emacs22_22.2-0ubuntu9_i386.deb
    • emacs22-bin-common_22.2-0ubuntu9_i386.deb
    • emacs22-common_22.2-0ubuntu9_all.deb
  • Run these commands:
    • sudo dpkg -i emacs22_22.2-0ubuntu9_i386.deb
    • sudo dpkg -i emacs22-bin-common_22.2-0ubuntu9_i386.deb
    • sudo dpkg -i emacs22-common_22.2-0ubuntu9_all.deb
  • Install hyperlatex with the command: 
        sudo apt-get install hyperlatex

Happy! Happy! Joy! Joy!
