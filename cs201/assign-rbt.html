The RBT tree module

Here is a conforming rbt.h file: 
    /*** RBT binary search tree class ***/

    #ifndef __RBT_INCLUDED__
    #define __RBT_INCLUDED__

The methods of the RBT class are similar to those of the GST class. As always, your implementations should equal that of a professional computer scientist.

You will store RBT values into a GST object. These values will store the generic values to be inserted into the red-black tree and the colors of the tree nodes.

To swap RBT values, a swapper function is needed: 
    function swapper(TNODE *a,TNODE *b)
        {
        void *va = getTNODEvalue(a);
        void *vb = getTNODEvalue(b);
        // swap the GST values
        setTNODEvalue(a,vb);
        setTNODEvalue(b,va);
        // the above swap swapped the colors,
        // but the colors should stay with the nodes,
        // so swap the colors back to the original nodes
        RBTVALUE *x = getGSTvalue(a);
        RBTVALUE *y = getGSTvalue(b);
        int color = x->color;
        x->color = y->color;
        y->color = color;
        }
The swapper function is passed to the BST via the setBSTswapper method.

The only local includes a RBT module should have are rbt.h, bst.h, and tnode.h.
