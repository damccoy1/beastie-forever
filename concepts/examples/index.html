The Quick Quiz System
____________________

An example of the quick quiz system, the basis for the Concept Inventories for STEM repository:
  • example.q: An example quiz with all kinds of wiki markup.
  • example.html: What the example quiz looks like when rendered into html5.
  • example.form: The interactive form of the example quiz. You can play around with selecting questions and generating PDFs.
  • markup.pdf: A reference guide for the wiki markup suitable for printing.
  • markup.html: A reference guide for the wiki markup.
  • markup.q: The wiki version of the reference guide.
  • index.q: The wiki version of this web page.
