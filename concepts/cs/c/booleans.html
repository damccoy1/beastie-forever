  1. Which of the following is not considered a false Boolean literal?

  1. Which of the following is not considered a true Boolean literal?

    2. '1'
    3. 1

  1. Which of the following is not a valid Boolean expression?

    4. x <= y
  1. Which of the following is a valid Boolean expression?
    1. x = y
    2. x += y
    3. x + y
    4. x != y
  1. Which operator returns false when two things are not equal?
    1. =
    2. !=
    3. ==

  1. Which operation always returns true when a is not less than b?
    1. a == b
    2. a != b

  1. Which operator returns true when two things are equal?
    1. ==
    2. !=

  1. What does the == operator do?

    2. gives a new value to a pre-existing variable
    3. creates (or updates) a new Boolean variable

Concept: short-circuiting

  1. T or F: C will evaluate the expression y % 2 == 0. 
        y = 4;
    x = 2;
    a = y < x || y % 2 == 0;
  1. T or F: C will evaluate the expression y % 2 == 0. 
        y = 4;
    x = 2;
    a = !(y < x || y % 2 == 0);
  1. T or F: C will evaluate the expression y % 2 == 0. 
        y = 4;
    x = 2;
    a = x < y || y % 2 == 0;
  1. T or F: C will evaluate the expression y % 2 == 0. 
        y = 4;
    x = 2;
    a = !(x < y || y % 2 == 0);
  1. T or F: C will evaluate the expression y % 2 == 0. 
        y = 3;

        y = 3;
    x = 4;

  1. T or F: C will evaluate the expression y % 2 == 0. 
        y = 3
    x = 4
    a = x < y && y % 2 == 0
