Concept: spotting variables
Students should be able to distinguish variables from constants

  1. What variable (if any) is defined in the following code? 
        int
    printHi(char *name)
        {
        printf("Hi there, %s\n",name)
        return 0;
        }
    1. "Hi there, %s\n"
    2. no variable is defined
    3. name
    4. int
  1. What variable (if any) is defined in the following code? 
        int
    show(void)
        {
        printf("Show me!\n");
        return 0;
        }
    1. show
    2. printf
    3. void
    4. no variable is defined
  1. What variables are present in the following code? 
        void
    printHi(char *name)
        {
        int count = 3;
        printf("Hi There, %s-%d\n",name,count);
        return;
        }
    1. just name
    2. printHi and name
    3. there are no variables in the code
    4. name and count
    5. void and printHi
  1. What variables are present in the following code? 
        void
    show(void)
        {
        printf("Hi There!\n");
        return;
        }
    1. there are no variables in the code
    2. show, me, and printf
    3. just void
    4. void and show
    5. just printf
  1. How many variables are defined in the following code? 
        char *greeting = "Hi There";

    void
    printHi(char *name)
        {
        printf("%s, %s\n",greeting,name);
        return;
        }
    1. two
    2. one
    3. three
    4. none
  1. How many variables are defined in the following code? 
        char *greeting = "Hi There,";

    void
    show(char *me,char *last)

        printf(%s, %s %s\n",greeting,name,last)
        return;
        }
    1. none
    2. one
    3. four
    4. three
    5. two
