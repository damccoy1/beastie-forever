Scope

Concept: vocabulary

  1. With respect to scope, another name for `visible' is:
    1. in scope
    2. out of scope
  1. With respect to scope, another name for `inaccessible' is:
    1. out of scope
    2. in scope
  1. With respect to scope, another name for `accessible' is:
    1. in scope
    2. out of scope
  1. T or F: Non-local variables, with respect to a particular scope, are those variables defined in that scope.
  1. T or F: Local variables, with respect to a particular scope, are those variables defined in that scope.

Concept: enclosing scopes

  1. T or F: Variables in an enclosing scope are in scope.
  1. T or F: Variables in an enclosed scope are in scope.
  1. T or F: The global scope is enclosed by all other scopes.
  1. T or F: Every other scope is enclosed by the global scope.
  1. T or F: Suppose in the body of function f, an if block defines a variable. The scope of if block encloses the scope of f.
  1. T or F: Suppose in the body of function g, an if block defines a variable. The scope of g encloses the scope of if block.

Concept: scopes and definitions

  1. T or F: The set of names defined in the scope of a function body includes the local variables.
  1. T or F: The set of names defined in the scope of a function body includes the non-local variables.
  1. With respect to a function body, the function name belongs to the:
    1. set of non-local variables
    2. set of variables that are out of scope
    3. set of local variables
  1. With respect to a function body, the function name is defined in:
    1. the scope of the function body
    2. the scope enclosing the scope that holds the function definition
    3. the scope that holds the function definition
  1. T or F: The set of names defined in the scope of a function body includes the name of the function.
  1. With respect to a function body, the formal parameters of the function belong to the:
    1. set of non-local variables
    2. same scope as the local variables
    3. the global scope
    4. set of variables that are out of scope
  1. With respect to a function body, the formal parameters of the function are defined in the:
    1. the scope that holds the function definition
    2. the scope enclosing the scope that holds the function definition
    3. the global scope
    4. the scope of the function body
  1. T or F: The set of names defined in the scope of a function body includes the formal parameters.
