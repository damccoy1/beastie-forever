  1. Does the function call match the function definition?

    ...

    1. no, there are too few formal parameters

    3. no, there are too many arguments

  1. Does the function call match the function definition? 
        int square(int x) { return x * x; }
    ...
    int a = 3;
    int result = square(a);

    2. no, there are too many arguments in the call
    3. no, the function call should pass a value, not a variable
    4. no, the function call should pass the variable x, not a
    5. yes
  1. Does the function call match the function definition? 
        int square(x) { return x * x; }
    ...
    int result = square(3,7);
    1. no, there are too many arguments in the call

    3. no, there are too many formal parameters in the definition
    4. yes
