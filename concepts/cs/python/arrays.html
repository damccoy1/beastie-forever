Arrays

Note: The Computer Science term list is usually considered short-hand for the data structure known as a linked-lists. What is commonly called a list in Python is actually a form of array (specifically, a dynamic array), so the following questions use the term array rather than list.

Concept: slices

  1. Consider the slice [a:b:c]. The value of a is:
    1. refers to the step size
    2. must be present
    3. excluded from the slice
    4. must be zero
    5. included in the slice
  1. Consider the slice [a:b:c]. The value of b is:
    1. must be present
    2. must be zero
    3. excluded from the slice
    4. included in the slice
    5. refers to the step size
  1. Consider the slice [a:b:c]. The value of c is:
    1. included in the slice
    2. must be zero
    3. excluded from the slice
    4. must be present
    5. refers to the step size
  1. Consider the following code: 
        a = [7,11,42]
    b = a[0:2]
    b[1] = 13
    print(a[1])
    What gets printed?
    1. 11
    2. 13
    3. 42
    4. 7
    5. nothing, the previous line has an error
  1. Which of the following slices does not generate a copy of the entire array?
    1. b = a[::]
    2. b = a[0:-1]
    3. b = a[:]
    4. b = a[:len(a)]
