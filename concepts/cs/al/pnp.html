Concept: N P -completeness

  1. To show that a problem A is N P -complete, one task is to:
    1. show A is not in N P .
    2. show A is in 𝒫.
    3. show A is in N P .
    4. show A is not in 𝒫.
  1. Suppose B is an N P -complete problem. To show that a problem A is N P -complete, one task could be:
    1. show a exponential time/space reduction from B to A.
    2. show a polynomial time/space reduction from B to A.
    3. show an exponential time/space reduction from A to B.
    4. show a polynomial time/space reduction from A to B.
  1. Another way of stating “a reduction from A to B” is:
    1. convert an algorithm for B to an algorithm for A
    2. convert an algorithm for A to an algorithm for B
    3. solve B-type problems with an algorithm for A
    4. solve A-type problems with an algorithm for B
