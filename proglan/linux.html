Installing the Submit System on Linux or Windows 10 Bash


In your terminal window, run the following commands: 
    cd
    wget beastie.cs.ua.edu/submit
    wget beastie.cs.ua.edu/set.email
    chmod +x set.email
    chmod +x submit
    mkdir bin
    mv submit bin/
Now run these commands PRECISELY as written: 
    wget beastie.cs.ua.edu/extra
    cat extra >> .bashrc
Make sure you use two greater-than symbols, not one. Now you need to set your email address by running the command (replacing your-crimson-email-name appropriately): 
    ./set.email your-crimson-email-name@crimson.ua.edu
Now close your terminal window and open up a new one. You should be good to go.
