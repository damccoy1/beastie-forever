  n^0.9*log(n) v.s. n [Visualized]
Posted by: daweil - 08-27-2018, 03:57 PM - Forum: Prerequisite exam - Replies (1)

The use of a graphing calculator can really help understand the growth rate of algorithms, along with proving that the constants do not matter. Personally, I use https://www.desmos.com/calculator

But when doing this, you must consider going to infinity. Plot f(x) = x on the website provided, than plot g(x) = x^0.9 * log(x). At first glimpse, and with zooming out a little, you'll believe that g(x) has a quicker growth rate. But if you zoom WAY out, you'll notice when x = 1 * 10^10, the function f(x) overcomes the growth rate of g(x).

  Installing Valgrind on Mac (High Sierra and above)
Posted by: daweil - 08-25-2018, 08:22 PM - Forum: Project 0 - Replies (3)

You have two options to installing Valgrind on Mac:

Option 1.) Get a real laptop, either Windows or Linux.

Option 2.) You will need the following installed:

XCode: You can get it in the App Store or at https://developer.apple.com/xcode/

You will also need to install Homebrew Mac (https://brew.sh/), this can be done by typing in terminal:

Code:
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
Note: You may need to agree to the license of xcode if you have not opened it yet.

If your Mac OS is below 10.13 (High Sierra), you should just be able to do "brew install valgrind" and be done from here on out.

If you have High Sierra and above, you'll need to do a little more work.
Once homebrew is installed, type:

Code:
brew edit valgrind

This will open up a file in vim, go down to the URL in "head" section and change:

Code:
From: https://sourceware.org/git/valgrind.git
To: git://sourceware.org/git/valgrind.git


Quickly check for an update on Homebrew by typing "brew update" in terminal.

Finally, install valgrind by typing:
Code:
brew install --HEAD valgrind
It will take a bit of time. If you come across errors, post in here.
