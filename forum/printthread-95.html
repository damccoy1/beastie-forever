Since we're always shifting when necessary with regard to insertCDA and removeCDA, is it even possible to have the front of the array start anywhere other than index 0?

RE: CDA Logic - lusth - 08-29-2018

Yes. Consider

Code:
items = newCDA();
insertCDA(items,0,x);
insertCDA(items,1,y);
insertCDA(items,2,z);
removeCDA(items,0);

At this point, index 0, from the user's point of view, is at index 1, from the internal view; the call getCDA(items,0) should return y. Index 0, in the internal view, is part of the unfilled portion.
