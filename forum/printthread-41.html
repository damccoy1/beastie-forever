Accounts on Nixie - lusth - 09-24-2018

Some of you have requested accounts on a linux server that has valgrind and gdb installed and working. That server is nixie.cs.ua.edu. Connect to nixie with the command:

ssh myNixieLogin@nixie.cs.ua.edu

Here is a script for uploading your project to nixie (replace my login name with yours):

Code:
#!/bin/bash
login=lusth
path=`pwd`
dir=`basename $path`
(cd ..; \
    tar cvfz project.tgz $dir; \
    cat project.tgz | ssh $login@nixie.cs.ua.edu "cat | tar xfvz -")


Name this file "push" and put it into your home directory's bin. Run chmod to make it executable:

chmod +x ~/bin/push

Then you can run push from your project directory. I HIGHLY suggest you setup passwordless logins to nixie. Don't come asking me for help on a nixie project without you doing this.

Also, use two windows, one on your development machine and one on nixie.
