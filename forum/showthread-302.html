According to the pseudo code on http://beastie.cs.ua.edu/cs201/insert-rb.html:
 
Code:
         if (uncle is red)
               {
               color parent black
               color uncle black
               color grandparent red
               x = grandparent
               }
 
Which means we'll have to color the root red then because it's the root color it back to black.
