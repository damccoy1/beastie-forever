RE: Assignment description - lusth - 08-28-2018

If I do n insertions in a row and then ask for the size, then I have done n+1 amount of work (n updates to size - one for each insertion - and one retrieval of size). Average that total work over all inserts, and you get an average of one unit of work per operation.
