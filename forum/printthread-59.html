Can Heaps have one node?

I ask this question because of questions like: "T or F: the largest value in a min heap can be found at the root"

Technically this could be true if the heap is one node. also,

"T or F: A binary tree with one node must be a heap"

This is true if it can be one node right?

RE: Heap Questions - JackODonohue - 09-09-2018

Heaps may have single node because heaps may be empty, andÂ insertion into an empty heap yields another heap of size 1. Therefore if a binary tree has a single node, it is a heap since heaps may be of size 1, and there are no children which may be greater than or less than the root with any ordering. The only time when the largest value in a min heap is at the root is when the heap is of size 1. In all other cases, this is false since there may be a duplicate which would mean that a smallest value does not exist. Alternatively, there may exist a distinct value which is less than the largest value, so if the largest value is at the root, the heap property is violated.
