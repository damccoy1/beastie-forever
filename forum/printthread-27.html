Keep in mind that everything inserted starts as red. That means the initial root is first red then recolored. Left rotation and 2 recolors after insertion of 3. No rotations and 4 recolors after insertion of 5. Then finally one double rotation and 2 recolors after inserting 4.

According to the pseudo code on http://beastie.cs.ua.edu/cs201/insert-rb.html:
 
Code:
         if (uncle is red)
               {
               color parent black
               color uncle black
               color grandparent red
               x = grandparent
               }
 
Which means we'll have to color the root red then because it's the root color it back to black.
