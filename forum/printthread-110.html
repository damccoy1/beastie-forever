Installing Valgrind on Mac (High Sierra and above) - daweil - 08-25-2018

You have two options to installing Valgrind on Mac:

Option 1.) Get a real laptop, either Windows or Linux.

Option 2.) You will need the following installed:

XCode: You can get it in the App Store or atÂ https://developer.apple.com/xcode/

You will also need to install Homebrew Mac (https://brew.sh/), this can be done by typing in terminal:
Code:
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
Note: You may need to agree to the license of xcode if you have not opened it yet.

If your Mac OS is below 10.13 (High Sierra), you should just be able to do "brew install valgrind" and be done from here on out.

If you have High Sierra and above, you'll need to do a little more work.
Once homebrew is installed, type:

Code:
brew edit valgrind

This will open up a file in vim, go down to the URL in "head" section and change:

Code:
From: https://sourceware.org/git/valgrind.git
To: git://sourceware.org/git/valgrind.git


Quickly check for an update on Homebrew by typing "brew update" in terminal.

Finally, install valgrind by typing:
Code:
brew install --HEAD valgrind
It will take a bit of time. If you come across errors, post in here.
