RE: Polynomialy Greater/Lesser - lusth - 09-25-2018

For example, n / log n grows more slowly than n, so it should be case 1, if anything. But there is no epsilon I can subtract from the exponent 1 so that n^(1-epsilon) still grows faster than n / log n. Thus this example lies between case 1 and case 2.
