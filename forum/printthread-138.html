RE: debugSTACK - lusth - 09-12-2018

A STACK debug level of one means the stack display function should call the displayDA with a DA debug level of zero.

A STACK debug level of two means the stack display function should call the displayDA with a DA debug level of one (i.e. a debugged display of the DA).

RE: debugSTACK - lusth - 09-14-2018

No. setSTACKdisplay should cache the display function (it needs it to print the elements of the DA in reverse order) and it should call setDAdisplay so that the underlying DA can display the elements for stack debug levels 1 and 2.
