I've created an animation of the building and solving of a maze and thought some may find it useful to visualize what is going on.  When building the maze, the 0 marks what cell you are inside.  The color of the 0 changes every time a wall is removed.  When the 0 backs up while staying the same color, that means there are no eligible neighbors from which to remove a wall, and the stack is popping until a cell with an eligible neighbor is found.
