So I see two ways a CDA can be implemented:

1. index 0 is connected to size on the other end: this way, the unfilled portion of the array always remains on the right, and only the filled portion actually becomes cyclic. Example with indices: |3|4|0|1|2| | |

2. index 0 is connected to capacity: this way, the unfilled portion can be in the middle of the filled portion. The above example would be implemented as: |3|4| | |0|1|2|

Which approach are we using for this project?
