Use the assert function from assert.h (Your first option). It should provide enough detail.

Example:
I have x = 5, and then assert(x == 7). Executing my program results in the following output:

ex: ex.c:10: main: Assertion `x == 7' failed.
Aborted (core dumped)

The assert call was in fact on line 10.
