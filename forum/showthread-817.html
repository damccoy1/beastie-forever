Creating a two-dimensional array of pointers to type X looks like this:

Code:
X ***array;

//create a one-dimensional array of row pointers
array = malloc(sizeof (X **) * rows);

//make all the rows
for (i = 0; i < rows; ++i)
   {
   //create a single row
   array[i] = malloc(sizeof(X *) * columns);
   //initialize the slots in the row
   for (j = 0; j < columns; ++j)
       array[i][j] = newX();
   }
