RE: Assignment description - lusth - 09-07-2018

I'm restricting the use of the random number generator so that we end up with the same mazes.

Use a new random number each time a choice of a wall removal is made (i.e. the next neighbor to pursue). But make sure you use just one.

You could generate a random number to pick a neighbor. If that neighbor was visited, you could generate a new random number to pick another neighbor and so on, until you get an unvisited neighbor. But I don't want you to do that.
