In the directions for "Tracing recursive functions", it states:

"When asked about the number of recursive calls, do not include the original call".

Just to be clear- does that mean when an if statement is looped through a total of 5 times, then 4 are considered recursive calls?

RE: Recursive Functions - magarwal - 09-09-2018

The original call is the function itself called from outside (like from main method).

RE: Recursive Functions - lusth - 09-09-2018

function f(n) { if (n < 2) return 1; else return n + f(n-1); }

function main() { println("f(5) is ",f(5)); }

If the above program is run, f is called five times: f(5), f(4), f(3), f(2), f(1), but only four of those calls are recursive calls.
