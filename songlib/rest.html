Printable Version

You can generate a period of silence with the rest function, which takes a beat count as an argument. For example: 

    rest(W);

would generate four beats of silence under 4:4 time. One can also rest for some number of measures, by using the measuresToBeats conversion function. For example: 

    rest(measuresToBeats(3));

would rest for three measures or, under 4:4 time, twelve beats.
